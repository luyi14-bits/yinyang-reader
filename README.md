## 🎯 风格说明

每篇都遵循这些规则：

| 要点 | 说明 |
|------|------|
| 开局即高能 | 第一句话钩住你，前三段进入恐怖 |
| 口语化叙事 | 像老辈人讲故事，不堆形容词 |
| 真实感 | 民俗细节有根有据，不是瞎编 |
| 逻辑恐怖 | 前因后果严丝合缝，结尾让你倒吸一口气 |
| 短平快 | 每篇 5 分钟，手机翻两页 |

写作标准详见 [`docs/writing-guide.md`](docs/writing-guide.md)  
写手 Skill：[`luyi14-horror-story-writer`](.trae/skills/luyi14-horror-story-writer/SKILL.md)
