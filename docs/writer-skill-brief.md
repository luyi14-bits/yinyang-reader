### 0.2 这五本书的共同基因

通过对五部爆款的读者评价进行关键词提取，得出下面这张"为什么好看"的底层原因表：

| 共性 | 含义 | 读者原话 |
|------|------|----------|
| **真实感压倒一切** | 不是编的，像真事。专业知识扎实，细节有根有据 | "作者据说是出马弟子""像真的经历过""代入感超强" |
| **开局即高能** | 前三章就要抓住人，不允许慢热 | "前三章直接劝退胆小鬼""开局就被活埋" |
| **不降智不套路** | 角色有正常智商，不靠无脑推进 | "没有无脑降智""主角是普通人不是超人" |
| **民俗知识内化** | 专业术语自然融入，不炫学 | "堂口、撞邪、破煞写得跟真事一样" |
| **短篇合集是趋势** | 碎片化阅读时代，一章一个独立故事最吃香 | 《中国民间灵异大事件》一章一个，和我们的方向完全一致 |
| **节奏密，爽点密** | 不给读者喘息的机会，读完一篇马上想下一篇 | "越看越上头""不敢关灯还想看" |

## 一、需求锚定（甲方原话逐句拆解）

| 甲方原话 | 拆解要点 |
|----------|----------|
| "AI写的，它没有那个逻辑感" | AI 只会接句子，不会接因果。逻辑感 = 前因后果的严丝合缝 |
| "没那个活人感" | AI 的人物像模板拼的，没有人的"毛边" |
| "它就是个文章而已" | AI 写的是"文本"，人写的是"故事" |
| "故事会那种短篇小说，两三页" | 1500-3000 字，翻三页纸的量 |
| "看的贼鸡巴带劲" | 节奏要紧，悬念要钩人，结尾要翻 |
| "恐怖故事，鬼故事" | 题材锁定灵异/惊悚/民间怪谈 |
| "上班看一看，爽的批爆" | 摸鱼场景，可碎片化阅读，高刺激 |

## 二、2025 恐怖爆款的六条血酬定律

> 核心判断：甲方要的不是老掉牙的故事会复刻，而是 **2025 年手机阅读时代** 的恐怖短篇。以下是当前市场验证过的六条铁律。

### 2.1 真实感是第一生产力

《鬼堂首徒》作者是出马弟子 → 读者买账。"真实感"不是"写实主义"，是**细节有根有据**：
- 堂口怎么设 → 不能瞎编
- 符怎么画 → 不能瞎编
- 撞邪什么反应 → 要懂的人看了点头

**操作指南**：每篇故事里的民俗知识至少有一个点是查证过的真实民俗，其他可以适度发挥。

### 2.2 开局钩子决定生死

《坊间诡术》男主被活埋乱葬岗——前三章就锁死了读者。

**操作指南**：钩子不是"标题党"，是正文第一句话就让人没法划走。
```
✅ "我被活埋的那天晚上，听见上面有人在唱歌。"
✅ "师父说，出马弟子不能收死人的钱。我收了。"
❌ "这是一个发生在2019年秋天的故事……"
```

### 2.3 短篇合集 > 系列连载

《中国民间灵异大事件》——一章一个独立故事，全网追疯。每篇独立、可碎片、随手打开就读，完美适配摸鱼场景。

**与我们的方向完全一致。**

### 2.4 "普通人主角"永远比"天选之人"好

《诡舍》9.7 分的关键：主角宁秋水是个普通人，前期也胆小也犹豫。"不是天生的英雄"= 代入感。

**操作指南**：视角人物永远有弱点。会害怕、会犹豫、会犯傻。读者不会代入英雄，但一定会代入一个和自己差不多的普通人。

### 2.5 节奏要"窒息感"

五部爆款的共同点：不给读者喘气。恐怖→缓和→更恐怖→再缓和→逼近临界点→突然爆发。缓和的时间越来越短，恐怖越来越密。

### 2.6 专业知识内化，不炫学

出马弟子的核心竞争力不是"知道得多"，是**把事说成"亲眼见过的"**。

**操作指南**：不写"据说出马弟子要……"，写"我见过师父摆堂口。先把香点上，三根一起点。香灰不掉，说明仙家来了。"

## 四、融合后的"恐怖短篇写手法则"

### 4.1 六条铁律（故事会 + 2025 融合版）

| # | 铁律 | 2025 增强 |
|---|------|----------|
| 1 | **有故事核** | 每篇必须有一个"一句话就能让朋友卧槽"的核心反转 |
| 2 | **结构完整简洁** | 500-2000 字，只讲一件事，配角用完就扔 |
| 3 | **语言口语化** | "像亲眼见过的人讲给你听"，禁止书面语 |
| 4 | **不抒情不议论** | 禁止任何"我感到""我明白""这件事告诉我" |
| 5 | **真实感**（新增） | 每篇至少一个查证过的真实民俗细节 |
| 6 | **开局即高能**（新增） | 第一句就要钩住人，前三段进入恐怖 |

### 4.2 四条反 AI 戒律

| 戒律 | AI 会做 | 你必须做 |
|------|--------|---------|
| 不堆形容词 | "夜色如墨""诡异氛围" | "天黑透了""连狗都不叫" |
| 不写心理活动 | "我感到一阵恐惧" | 用身体反应替代——手心出汗、后背发凉 |
| 不总结升华 | "从那以后我明白……" | 直接收，留白 |
| 不让人物说一样的话 | 所有角色一个口吻 | 师父话少语沉，徒弟嘴笨犯傻，路人带方言 |

## 五、叙事结构（五段式 × 2025 节奏感）

### 5.1 五段式结构

```
【钩子】开头 50 字以内
一句话抓人。用一句老规矩、一个反常现象、一个不可挽回的行动。
✅ 2025 级句式：
  "我被活埋的那天晚上，听见上面有人在唱歌。"
  "师父说，出马弟子不能收死人的钱。我收了。收了三万。"
  "我开 13 路末班车三年了。每个月的十五号，都会上来一个不该上来的人。"

【铺陈】100-300 字
只交代必须知道的。人物+场景+第一个不对劲的信号。
- 视角人物出场（有弱点的普通人）
- 时间地点（两三句）
- 第一个异常细节（自然带出，不渲染）
禁止：大段环境描写、人物生平

【升温】400-1000 字（占全文 50%+）
恐怖逐层展开。2025 新要求——节奏要"密"：
- 异常→短暂缓和→更异常→更短缓和→逼近临界点
- 缓和的时间越来越短
- 恐怖越来越密集
- 每段一个信息量，每个细节后面都用得上

【高潮】200-500 字
反转/爆发。必须是升温段里长出来的。
读者回看前面会拍大腿的技术。

【收尾】50 字以内
一句收。平淡但越想越毛。
✅ "后来我再也没开过那条线路。但每个月十五号，
     调度室还是会接到一个电话。问我为什么改线了。"
✅ "猪肉还是热的。"
```

### 5.2 字数配比

| 构件 | 1500字 | 2000字 |
|------|--------|--------|
| 钩子 | ~50字 | ~50字 |
| 铺陈 | ~200字 | ~300字 |
| 升温 | ~800字 | ~1100字 |
| 高潮 | ~350字 | ~450字 |
| 收尾 | ~50字 | ~50字 |

## 六、恐怖技法工具箱

### 6.1 恐惧三层次

| 层次 | 说明 | 2025 中式案例 |
|------|------|-------------|
| 恶心/血腥 | 慎用 | 腐烂、断肢 → 用也一笔带过 |
| **恐怖** | 中层，对反常的惊骇 | 纸人转头、堂口桌上多了一副碗筷、监控里拍到不该拍的 |
| **惊惧** ⭐ | 最深层，对不可知的战栗 | 收了三万块钱的那个人，三年前就死了——那转账的人是谁？ |

### 6.2 六种最有效的叙事技法

| 技法 | 说明 | 2025 示例 |
|------|------|----------|
| 细节恐怖 | 不是大事，是一次呼吸、一个眼神 | "他说话的时候，嘴唇没动" |
| 数量恐怖 | 多一个、少一个 | "堂口上只摆了五碗水，但我数出六碗" |
| 重复恐怖 | 同样的事反复，越来越近 | 每个月十五号那个电话 |
| **逻辑恐怖** ⭐ | 因果推出无法接受的结论 | 收了死人的钱，退回去的时候发现——打款账户是三年前注销的 |
| 留白恐怖 | 不说满 | "他没回答。摆了摆手：赶路。" |
| 日常恐怖 ⭐ | 发生在最普通的地方 | 公交末班车、出租屋、堂口、医院电梯 |

### 6.3 2025 新增叙事技法

| 技法 | 说明 | 出自 |
|------|------|------|
| **单元剧滚动** | 每篇独立，共用同一个"我"的世界 | 《13路末班车》《中国民间灵异大事件》 |
| **职业视角** | "我"有一个特殊职业（司机/出马弟子/扎纸匠），用职业自然引出恐怖 | 《13路末班车》《鬼堂首徒》 |
| **开局不可逆** | 不是"遇到怪事"，是"已经做了不可挽回的事" | 《坊间诡术》被活埋 | 
| **知识即悬念** | 用专业民俗知识本身制造恐怖——知道规则的人比不知道的更害怕 | 《鬼堂首徒》 |

## 七、AI vs 活人：核心差距

| 维度 | AI 特征 | 活人特征 | 甲方痛点 |
|------|---------|----------|----------|
| 人物 | 所有角色说话一样 | 每人有"语言指纹" | "没活人感" |
| 因果 | 接句子，不接因果 | 前因后果严丝合缝 | "没逻辑感" |
| 情绪 | 靠形容词堆砌 | 靠细节传递 | "就是个文章" |
| 节奏 | 匀速推进 | 该快该慢，有呼吸感 | "不带劲" |
| 结尾 | 总结式收尾 | 留白/反转/似懂非懂 | "不像人写的" |
| **真实感** | 数据库拼贴 | 有活人才能有的细节 | **新增** "不真实" |

### 具体案例

```
❌ AI：
"夜色如墨般浓重，寂静的村庄笼罩在一片诡异的氛围之中。我心中
涌起一股莫名的恐惧，总觉得有什么看不见的东西正隐藏在黑暗中……"

✅ 2025 爆款写法：
"天黑透了。村口连狗都不叫。
我走到堂口前，香已经烧了半截。师父说过，香烧到一半不能灭。
灭了就得跑。
我抬头看。香灭了。
三根全灭。"
```

## 八、场景与素材池

### 8.1 场景池（× 2025 扩展）

| 场景 | 恐怖来源 | 2025 新增变体 |
|------|----------|-------------|
| 公交/末班车 | 密闭空间+陌生人 | 每晚同一个时间同一个站上来一个不该上来的人 |
| 堂口/出马 | 专业民俗 | 香灭了、仙家来了但来的不是该来的 |
| 医院深夜 | 生死边界 | 太平间灯自己亮、空病房的呼叫铃 |
| 乱葬岗/坟地 | 死亡的物理在场 | 不是"看见鬼"，是"被埋进去" |
| 出租屋 | 你以为一个人 | 合租室友不存在、上一任租客还住着 |
| 电梯 | 垂直封闭 | 每层停但没人、监控拍到背后有东西 |
| 拆迁区 | 新旧缝隙 | 拆了一半的楼亮着灯 |
| 深夜便利店 | 都市孤岛 | 同一个客人买了两次同一件东西 |

### 8.2 角色池

| 角色 | 功能 | 2025 增强 |
|------|------|----------|
| 普通人"我" | 视角代入 | ✅ 有职业身份（司机/扎纸匠/出马弟子），不是为了写故事随便捏的人 |
| 懂行的 | 揭示真相 | ✅ 话少、见过世面、说一半留一半（如师父/老保安/姥姥） |
| 不信邪的 | 对比衬托 | ✅ 嘴硬，后来出事了——但死得不敷衍 |
| 当事人 | 故事核心 | 亲历者，从他嘴里说出最恐怖的细节 |

### 8.3 中式恐怖素材池（含出马/堂口）

| 素材 | 说法 | 用法 |
|------|------|------|
| 纸人纸马 | 不烧必招祟 | 扎纸铺、祭祖忘了烧 |
| 三把火 | 两肩+头顶 | 走夜路不能回头 |
| 头七 | 回魂夜 | 死者那晚真的回来了 |
| 替死鬼 | 找替身 | 河边/桥上/路口 |
| **出马/堂口** ⭐ | 仙家上身 | 香灭了、仙家不是该来的那个、请神容易送神难 |
| **收死人的钱** ⭐ | 拿了不该拿的 | 转账/红包/压在枕头下的钱 |
| 喊魂 | 回头就被带走 | 半夜有人叫你别答应 |
| 镜子 | 午夜不照镜 | 宾馆镜子对床 |
| 门槛 | 挡外不挡内 | 门槛下压了东西 |
| **末班车** ⭐ | 最后一班不是给人坐的 | 多了一个人/少了一个站/时间不对 |

## 九、语言典范

### 9.1 应该写的句子

```
"天黑透了。"
"香灭了。三根全灭。"
"他说话的时候，嘴唇没动。"
"猪肉还是热的。"
"她笑了。嘴咧到了耳根。"
"调度室又来电话了。问我还开不开 13 路。"
```

### 9.2 绝不写的句子

```
❌ "夜色如墨般浓重，空气中弥漫着不祥的预感。"
❌ "我感到一阵前所未有的恐惧从心底涌起。"
❌ "这一切的背后，究竟隐藏着怎样的真相？"
❌ "从那以后，我明白了一个道理……"
❌ "他的眼睛像两潭深不见底的寒水，藏着无尽的哀伤。"
```

### 9.3 节奏控制

| 场景 | 句长 | 分段密度 |
|------|------|----------|
| 日常铺陈 | 15-20字 | 正常 |
| 发现异常 | 5-10字 | 增加 |
| 紧张时刻 | 3-5字，一句一段 | 高密度 |
| 师父/懂行的说话 | 中短，不解释 | 单独成段 |
| 结尾 | 极短，一段一句话 | 独立 |

## 十一、给 Skill 创建者的最终建议

### 11.1 写前自检（7 条）

- [ ] 故事核能一句话讲给别人且让人"卧槽"吗？
- [ ] 第一句话能钩住人吗？（用"老规矩+我破了"的句式）
- [ ] 里面有活人才写得出的细节吗？（不是数据库拼贴）
- [ ] 人物各自有"语言指纹"吗？
- [ ] 升温段埋的每个细节高潮都用上了吗？
- [ ] 结尾是反转/留白吗？（不是总结升华）
- [ ] 至少一个民俗知识点是查证过的真实民俗吗？

### 11.2 开局钩子速查（直接套用）

```
模板 1：老规矩+我破了
"师父说，出马弟子不能收死人的钱。我收了。"
"赶尸有三不赶。我师父三条全犯了。"

模板 2：已经做了不可挽回的事
"我被活埋的那天晚上，听见上面有人在唱歌。"
"我把那间房租出去了。租给了一个我从来没见过的人。"

模板 3：规律+异常
"我开 13 路三年了。每个月的十五号，调度室都会收到一个投诉。
投诉的人说——我少停了一站。但我的行车记录仪里，每一站都停了。"
```
