## 一、出马仙·堂口体系

### 1.1 核心概念

| 术语 | 释义 | 来源 |
|------|------|------|
| 堂口 | 供奉仙家、沟通灵界的场所，设在出马弟子（弟马）家中 | 东北民俗 |
| 弟马/香头/香童 | 仙家在人间办事的人，北京叫"香头"，东北叫"顶仙"/"顶香的" | 头条文章 |
| 出马 | 仙家"选中"某人，通过他给人看事、治病、解灾 | 民间传承 |
| 立堂 | 正式设立堂口，书写堂单，安置香炉，通常需要师傅指导 | 出马仙网 |

### 1.2 四大仙家（胡黄常蟒）

| 仙家 | 本体 | 角色定位 | 性格特征 | 主管事务 |
|------|------|----------|----------|----------|
| **胡仙** | 狐狸 | 军师/智慧担当 | 道行最深、最稳重 | 人脉、姻缘、指点方向 |
| **黄仙** | 黄鼠狼 | 传令兵/沟通担当 | 机敏、反应快 | 传递信息、感应人心、报信 |
| **常仙** | 蛇 | 守护神/镇宅担当 | 性情刚烈、气场强 | 镇宅、驱邪、挡煞 |
| **蟒仙** | 蟒蛇 | 武将/力量担当 | 体型大、威严 | 化解大灾大难、震慑邪祟 |

> 📌 **来源依据**：头条文章《揭秘出马仙堂口真相：胡黄常蟒谁当家，各自管什么事》

### 1.3 堂口陈设与规矩

| 条目 | 细节 |
|------|------|
| **堂单** | 书写仙家名号，挂在堂口正中 |
| **香炉** | 铜质圆形为佳（代表圆满），至少一个，通常五个（代表五路兵马） |
| **供奉** | 水果、酒、水、鲜花 |
| **铁律** | ① 医不叩门，道不轻传——仙家不会主动找人立堂 ② 香灭了有讲究 ③ 仙家不能强迫人 |
| **警惕** | 民间有骗子借"立堂口"高价收费，涉嫌诈骗 |

### 1.4 故事应用方向

| 切入点 | 恐怖逻辑 | 参考爆款 |
|---------|----------|----------|
| **收死人的钱** | 弟马收了不该收的红包/转账，退回去发现账户已注销 | 《鬼堂首徒》 |
| **仙家不是该来的那个** | 请来的仙家不对——黄仙装胡仙，或者混进了野仙 | 《鬼堂首徒》 |
| **香灭了** | 上香时香火突然断了/香灰打卷/三长两短 | 传统堂口禁忌 |
| **请神容易送神难** | 不该立堂的人硬立，招惹了不该惹的东西 | 民间传说 |
| **出马弟子破戒** | 弟子做了仙家不让做的事，仙家走了，东西来了 | 民间说法 |

## 二、扎纸匠规矩

### 2.1 三条铁规矩

| # | 规矩 | 原因 | 来源 |
|---|------|------|------|
| ① | **不给横死的人扎纸人** | 横死的人怨气重，纸人容易"沾气" | 头条文章《老扎纸匠的规矩》 |
| ② | **不给没成亲的年轻女子扎纸人** | 未婚夭折去了阴间要配阴婚，扎纸人容易招来不干净的东西 | 同上 |
| ③ | **扎纸人绝不点睛** | 点睛是造物的本事，只有老天爷和阎王爷能做；凡人点了睛，纸人就有了魂，要出大乱子 | 同上 |

### 2.2 其他细节

- 纸人做好后，会在纸人的耳朵上做记号（防邪物附身）
- 有些师傅还会在纸人背后写符
- 出殡时纸人烧得旺，火焰发蓝，说明亡魂笑着走了
- 扎纸匠的笔——"落下去的是纸，提起来的是命"

### 2.3 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **徒弟破戒点睛** | 学徒手痒给纸人点上眼睛，纸人活了/转头了/走了 |
| **红嫁衣纸人** | 横死的姑娘穿着大红嫁衣，纸人点了睛——借怨气出来喊冤 |
| **纸人多了/少了** | 明明只扎了两个，半夜一数多了一个；或者烧的时候少了一个 |
| **纸人在不该在的地方** | 纸人出现在灵堂之外——你家客厅、你的床头、你身后的公交座位上 |
| **纸人没烧干净** | 烧完发现纸人的半张脸还在灰里，眼睛还睁着 |

## 三、十字路口烧纸钱

### 3.1 规矩细节

| 条目 | 具体做法 | 来源 |
|------|----------|------|
| **时间** | 必须天黑以后，阴间的人只有晚上才出来活动 | 头条文章 |
| **画圈** | 在地上画一个圈，把纸钱围起来——不画圈的话谁都可以来拿 | 同上 |
| **圈外留钱** | 拿几张纸钱放在圈外边，给那些没有亲人烧纸的孤魂野鬼 | 同上 |
| **念叨** | 烧的时候要念叨"谁谁谁给你送钱来啦"，祖先能收到 | 同上 |
| **地点含义** | 十字路口四通八达，可沟通阴阳 | 民间信仰 |

### 3.2 延伸说法

- 烧错了时辰→钱收不到，反而招来不该来的
- 烧给了不该给的人→有东西跟着你回家
- 画圈的时候留了缺口→钱被抢光
- 路边看到烧过的纸灰→不要踩，不要跨

### 3.3 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **烧错了对象** | 在路口烧纸，念叨错了名字/没念叨——有东西跟回来了 |
| **圈外的东西** | 圈外放的纸钱被谁拿走了？拿钱的那个，跟在你后面 |
| **烧完发现不对劲** | 纸灰的颜色不对/风往不该吹的方向吹/路口多了一个人 |
| **有人在你烧过的地方烧** | 第二天同一个位置有新的纸灰——谁用的你的圈？ |

## 四、叫魂/喊魂

### 4.1 民间说法

| 条目 | 说法 | 来源 |
|------|------|------|
| **叫魂原因** | 人受到惊吓或生病，魂魄离散，必须叫回来，否则会慢慢病死 | 彝族/汉族共有民俗 |
| **做法** | 三里路，边走边喊，不能停，不能回头 | 头条文章 |
| **禁忌** | 叫魂的时候不许任何人跟着 | 民间故事《叫魂灯》 |
| **回头后果** | 一回头魂就散了，再也叫不回来；或者叫回来的是别的东西 | 民间传说 |

### 4.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **叫魂的人回了头** | 叫到一半忍不住回了头——看到身后不是路，是别的什么 |
| **叫回来的人不对劲** | 名字喊对了，人醒过来了，但是……眼神不对/说话不对 |
| **不该叫的时候叫了** | 中元节/子时叫魂——叫来的不是你丢的那个 |
| **有人半夜叫你名字** | 别答应。一答应魂就被勾走了 |

## 五、湘西赶尸

### 5.1 三大铁律

| # | 规矩 | 原因 | 来源 |
|---|------|------|------|
| ① | **符咒不能揭，铜钱不能掉** | 尸体入殓前额头贴镇尸符，嘴里含压魂钱——掉了尸身失控，魂灵散了，回不了故土 | 头条文章 |
| ② | **铃响尸走，铃停尸立。夜里赶路，白日住破庙义庄** | 赶尸人只许在前头摇引魂铃，铃声不断尸身才会跟着走；不能住正经客栈 | 同上 |
| ③ | **绝对不能回头，不能让人看见尸体的脸** | 无论身后有什么动静——有人喊名字、有东西碰衣角——都不能回头 | 同上 |

### 5.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **赶尸人破了戒** | 忍不住回头看了一眼——身后跟着的到底是不是尸体？ |
| **尸体少了/多了/换了** | 天亮一数，少了一具/多了一具/脸不一样了 |
| **住进了赶尸人住过的义庄** | 半夜听到铃铛声 |
| **符咒掉了** | 额头上的符被风吹走了，尸体停在了不该停的地方 |

## 六、午夜镜子禁忌

### 6.1 民间说法

| 条目 | 说法 | 来源 |
|------|------|------|
| **子时照镜** | 23:00-1:00 是一天中阴气最盛的时刻，人间能与另一个位面相通 | 头条文章 |
| **后果** | 镜中的倒影可能不是你自己，而是另一个位面的"徘徊者" | 同上 |
| **宾馆镜子** | 宾馆的镜子不能对床——半夜醒来会看到镜子里有人看着你 | 民间说法 |
| **梳妆台** | 老式梳妆台的镜子不用的时候要盖起来 | 民间习惯 |

### 6.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **镜中人不对** | 你笑他不笑/你抬手他没抬/你转身他还在看你 |
| **镜子里的世界** | 镜子里映出的不是你现在所在的房间——是另一个地方 |
| **镜子里多了一个** | 明明只有你一个人，镜子里有两个 |
| **凌晨三点** | 老说法——凌晨三点是阴气最重的时候，那时照镜子会看见不该看见的 |

## 七、门槛禁忌

### 7.1 民间说法

| 条目 | 说法 | 来源 |
|------|------|------|
| **别踩门槛** | 踩别人家门槛"断家运、挡财运" | 头条文章 |
| **别坐门槛** | 门槛是挡外不挡内的，坐门槛就是把家宅的运气压住了 | 民间说法 |
| **门槛下压东西** | 老房子门槛下常压着铜钱/符咒/桃木——镇宅用的 | 民间习俗 |
| **门槛的功用** | 门槛是用来挡邪祟的——邪祟不能跨门槛，只能跳，而门槛有讲究 | 民间传说 |

### 7.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **门槛下的东西被挖了** | 装修/翻新挖出了门槛下的东西——镇的东西放出来了 |
| **门槛自己坏了** | 门槛无故开裂/门槛上有水渍/门槛上有脚印——外面的东西进来了 |
| **跨不过门槛的人** | 客人走到门口，脚抬不起来——门槛不让他进/他不是人 |
| **门槛坐不得** | 有人偏要坐门槛——坐了之后家里开始不对劲 |

## 八、地基主

### 8.1 民间说法

| 条目 | 说法 | 来源 |
|------|------|------|
| **地基主是什么** | 每栋房子都有，是"土神"或"宅神"，保家宅平安 | 民间信仰 |
| **什么时候拜** | 搬入新家当天下午/安放神位/过年/元宵/清明/端午/中元/中秋/除夕 | 搜狐文章 |
| **怎么拜** | 在厨房或后门朝屋内拜，供品简单（饭菜、水酒） | 传统习俗 |
| **为什么要拜** | 地基主是房子的原主，不拜他不高兴 | 民间说法 |
| **不是跟社区一起拜** | 拜地基主是按"门牌号码"拜的，不是跟着社区大楼一起拜 | 山水有清音 |

### 8.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **没拜地基主** | 搬进新家没拜——半夜有人敲墙/水管响/总觉得有人在客厅 |
| **地基主不是原主** | 这块地以前是什么？坟地/乱葬岗/刑场——地基主不止一个 |
| **地基主生气了** | 供品没放对/忘记拜了/拜了错的——家里的东西开始移位 |
| **老宅的地基主** | 老房子拆了盖新楼，地基主还住在原来的位置——新楼的地下层 |

## 九、三把火（走夜路）

### 9.1 民间说法

| 条目 | 说法 |
|------|------|
| **三把火位置** | 左右肩膀各一把，头顶一把 |
| **走夜路禁忌** | 不能回头——一回头，肩上的火就灭了，脏东西就能靠近你 |
| **被拍肩膀** | 有人拍你肩膀——千万别回头。拍了肩膀火就灭了 |
| **火灭了以后** | 脏东西会跟着你，或者直接上你的身 |

### 9.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **走了夜路回了头** | 有人叫你名字——你回头了——身后什么都没有——但从那天起你总觉得有人在背后 |
| **被拍了肩膀** | 夜路上有人从后面拍你——你回头了——拍你的人长什么样？你没看见，因为你回头的时候他就在你身后 |
| **头顶火灭了** | 走了很长很长的夜路，头顶的灯/帽子掉了——第二天开始你像变了个人 |

## 十、头七与回魂

### 10.1 民间说法

| 条目 | 说法 |
|------|------|
| **头七** | 死后第七天，亡魂会回到家中最后一次看望亲人 |
| **准备** | 家里要摆一桌饭菜（死者生前爱吃的），家人要回避 |
| **撒灰** | 门口/地上撒一层灰或面粉——第二天早上看有没有脚印 |
| **规矩** | 头七晚上不要出门/不要叫名字/不要照镜子 |
| **为什么回避** | 亡魂看到亲人会舍不得走，影响投胎 |

### 10.2 故事应用方向

| 切入点 | 恐怖逻辑 |
|---------|----------|
| **灰上的脚印** | 第二天看灰上的脚印——不是人的，或者方向是朝外的——走出去的没走回来 |
| **饭被吃了** | 头七摆的饭菜真的少了——有人回来吃了 |
| **有人没回避** | 头七那晚有人没走——看到了回来的"人"——回来的不是死者 |
| **头七之后** | 头七过了，但东西还是在动——他没走 |

## 十一、其他零散禁忌

| 禁忌 | 说法 | 故事应用 |
|------|------|----------|
| **猫跳过棺材** | 猫通灵，猫跳过棺材会让尸体"诈尸" | 守灵时猫进来了/猫跳上去了 |
| **黑狗对着空墙角叫** | 黑狗能看到人看不到的东西 | 半夜狗对着你家墙角叫——墙角啥也没有 |
| **烧纸画圈留缺口** | 圈要留一个朝向死者坟墓方向的缺口，否则钱送不出去 | 缺口留错方向了/没留缺口——钱没送到，死者托梦了 |
| **孕妇不进灵堂** | 孕妇身上有两条命，容易冲撞/被盯上 | 孕妇不得已进了灵堂——从那以后胎像不稳 |
| **雨伞不带进别人家** | "伞"谐音"散"，且伞是给阴间人遮阳的 | 下雨天带伞进了亲戚家——雨伞放在门边——伞被人收起来了 |
| **夜里不剪指甲** | 夜里剪指甲，魂魄会被剪掉/鬼会跟着你 | 半夜剪指甲——每剪一下，外面就有东西敲门 |
| **不再穿死者生前的鞋** | 死人穿过的鞋，活人穿了会走他的路 | 贪便宜穿了死者留下的鞋——走路的姿势越来越像那个人 |
| **捡到的钱不能花** | 十字路口捡到的钱可能是烧给死人的"买路钱" | 捡了路口的一块钱——从那以后每天晚上有人找你找钱 |

## 十二、故事应用索引

### 12.1 场景 × 恐怖素材 交叉表

| 场景 | 推荐素材 | 推荐视角 |
|------|----------|----------|
| 深夜医院 | 太平间、空病房呼叫铃、地基主（医院建在什么地上） | 护士/护工/保安 |
| 老式居民楼 | 门槛、地基主、楼道灯坏、对门无人却有电视声 | 住户/外卖员/快递员 |
| 出租车/网约车 | 末班车公式、后视镜多了一个、乘客不是人 | 司机 |
| 电梯 | 每层停但没人、镜中人不对、多了一个人 | 住户/保安 |
| 拆迁区/城中村 | 地基主（拆了的房子地基还在）、纸人（拆迁户烧纸） | 拆迁工人/记者 |
| 出租屋 | 上一任租客、镜子对床、半夜有人叫名字 | 租客 |
| 深夜便利店 | 同一个客人进了两次、买纸钱的客人、收银机多找的钱 | 店员 |
| 农村老宅 | 门槛、头七、扎纸匠、三把火、出马堂口 | 返乡青年 |
| 殡仪馆/灵堂 | 纸人、猫、烧纸、守灵 | 殡仪馆工作人员/家属 |
| 山间夜路 | 三把火、叫魂、赶尸 | 路过的人/赶尸人 |

### 12.2 恐惧层次分类

| 层次 | 对应素材 | 用法要点 |
|------|----------|----------|
| **恶心/血腥**（慎用） | 纸人点睛、尸体符咒掉了 | 一笔带过，不靠这个取胜 |
| **恐怖**（中层） | 镜中人不对、纸人转头、楼梯多了一级 | 制造当场惊吓 |
| **惊惧**（核心目标） | 死后七年的人三年前回来过、账户已注销但钱刚转出去、鞋为什么每年都来 | 关掉屏幕还在想 |

### 12.3 "一环扣一环"升温模板

> **通用节奏**：
> 一个异常 → 短暂缓和 → 更异常 → 更短缓和 → 逼近临界点

**示例——纸人篇**：
```
学徒扎纸人 → 手痒点了睛（异常）→ 纸人没动静，他松了口气（缓和）
→ 半夜醒来，纸人不在桌上（更异常）→ 找了一圈，在门口找到的——可能是自己记错了（更短缓和）
→ 第二天发现纸人站的位置变了/面朝的方向变了（逼近临界点）
→ 第三天晚上，纸人站在他床头（高潮）
```

**示例——出马篇**：
```
弟马收了香客三万块钱（异常）→ 香客说事办好了，满意走了（缓和）
→ 弟马想把钱转存，发现转账的账户已注销三年（更异常）
→ 查香客的名字——三年前就死了（更短缓和）
→ 那三万块钱现在在哪儿？弟马低头看钱包——现金，崭新崭新的人民币，编号全是连着（逼近临界点）
→ 手机震了：香客发来一条微信："钱收到了吗？我到了。"（高潮）
```
