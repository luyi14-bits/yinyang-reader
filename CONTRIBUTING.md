# 贡献指南

感谢你对本项目的关注！欢迎通过以下方式参与。

## 📖 内容贡献

本项目以志怪/恐怖短篇小说为主。如果你有合适的稿件：

1. 查看 `docs/writing-guide.md` 了解风格要求
2. 将稿件以 Markdown 格式保存到 `stories/` 目录
3. 更新 `stories/INDEX.md` 添加索引条目
4. 如果使用在线阅读器，同步更新 `stories/` 下的 JSON 数据文件

## 🐛 报告问题

如果发现 Bug 或有改进建议，请通过 GitHub Issues 提交，并提供以下信息：

- 问题描述
- 复现步骤
- 截图（如适用）
- 浏览器版本（如适用）

## 💻 开发贡献

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feat/your-feature`
3. 提交更改
4. 推送到你的分支
5. 提交 Pull Request

## 📐 编码规范

- 保持简洁，避免过度抽象
- 使用语义化的 commit message
- 保持向后兼容

## 📄 许可

提交代码即视为同意 [LICENSE](LICENSE) 许可协议。
