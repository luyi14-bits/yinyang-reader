# 安全策略

## 支持的版本

本项目当前正处于开发阶段（v0.x），以下版本提供安全更新：

| 版本 | 支持状态 |
|------|----------|
| v0.x | ✅ 活跃开发中 |
| < v0.1 | ❌ 不支持 |

## 报告安全漏洞

如果你发现安全漏洞，请**不要**通过公开的 GitHub Issues 报告。

请直接联系项目维护者，我们将尽快处理。

## 安全最佳实践

### ⚠️ 第三方字体

本项目的 `reader.html` 使用了 Google Fonts（Noto Serif SC、Ma Shan Zheng）。如果部署在内网环境，建议将字体自托管以避免外部请求。

### ⚠️ PWA 安全

使用 Service Worker 时请注意：

1. Service Worker 仅在 HTTPS 或 localhost 下生效
2. 确保 `sw.js` 不被篡改
3. 定期更新缓存策略

### ⚠️ 内容安全

本项目为纯静态内容，不涉及用户数据收集。故事内容均为原创或经过授权的改编。

## 依赖管理

- 无外部运行时依赖（纯静态 HTML/JS）
- 构建脚本使用 Node.js 标准库，无需额外安装
